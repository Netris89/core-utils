# True

Simple implementation of a POSIX true command-line utility in C++,

## Build Instructions
> [!IMPORTANT]
> Requirements: CMake 3.10+, a C++20-compliant compiler (e.g., GCC 10+, Clang 9+).
```sh
git clone https://github.com/Netris89/true
mkdir build
cd build
cmake ..
cmake --build .
```

> [!NOTE]
> More details on the true command and its behavior can be found here:
> [The Open Group - true utility](https://pubs.opengroup.org/onlinepubs/9799919799/utilities/true.html)
